Install BWA

```
cd ~
curl -L https://sourceforge.net/projects/bio-bwa/files/bwa-0.7.15.tar.bz2/download > bwa-0.7.15.tar.bz2

tar xjvf bwa-0.7.15.tar.bz2
cd bwa-0.7.15
make
 
export PATH=$PATH:$HOME/bwa-0.7.15/
```
